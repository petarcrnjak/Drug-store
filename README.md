Student project of the college "Web design and Programing".
1. Description

Drug store is a web application that allows users to browse and order medicines. There are also different types of users, such as unregistered, 
registered users, moderators, administrators, and a variety of their responsibilities. Implemented functionalities are login, registration, reservation of 
medicines, my account, virtual time configuration, history tracking and more.

2. Solution

The application is a simple web-based system built using HTML and connected to a database.

3. Used technologies and tools

Technologies which are used are HTML and CSS, there are also PHP, MySQL and JS.
The development was done using NetBeans IDE for creating code and editing, MySQL Workbench for database managment and creating a ERA model.
Web tool Gliffy (https://www.gliffy.com) for creating a navigation diagram.
